Engine Documentation

A lightweight, modular 2D game engine built with TypeScript and Canvas API

Fixed-Timestep Game Loop

Deterministic physics with configurable timestep (default 60 FPS). Prevents spiral of death with delta clamping and accumulator-based updates. Built-in FPS counter and frame time tracking.

Scene Management

Stack-based scene system with lifecycle hooks (onEnter, onExit, onResize). Automatic context injection with renderer, input, camera, and physics. Easy scene transitions and state management.

2D Camera System

World-to-screen and screen-to-world transforms. Smooth pan/zoom with mouse support. Viewport culling via getWorldBounds(). Configurable min/max zoom limits (0.01 to 50x).

Collision Detection

Full SAT (Separating Axis Theorem) implementation. Supports Circle, AABB, OBB, Polygon, and Capsule shapes. All shape pairs supported with contact point generation and penetration depth.

Rigid Body Physics

Full rigid body dynamics with linear/angular velocity. Force, impulse, and torque accumulation. Material properties (restitution, friction, damping). Static, kinematic, and dynamic body types. Auto-sleep optimization.

Collision Shapes

CircleShape, AABBShape, OBBShape (oriented box), PolygonShape (convex), CapsuleShape. Automatic inertia computation. World-space vertex/normal transforms for SAT tests.

Particle System

Object-pooled particles with zero GC pressure. Emission shapes: point, circle, rect, line. Velocity cones with spread. Color/size gradients over lifetime. Gravity, drag, and blend modes.

Sprite System

SpriteSheet loading with named frame definitions. defineRow() for animation strips. AnimatedSprite with FPS control and loop/one-shot modes. TiledSprite for parallax scrolling backgrounds.

Lighting System

Dynamic 2D lighting via off-screen compositing. PointLight with radial falloff. SpotLight with cone angle and direction. DirectionalLight for ambient. Configurable color, intensity, and inner radius.

QuadTree

Generic spatial partitioning for broad-phase collision. Rectangle and circle queries. Nearest neighbor search. Barnes-Hut gravity approximation for N-body simulations (O(n log n) complexity).

TileMap System

Multi-layer tile rendering with visibility control. Tile types: road, sidewalk, grass, building. Solid/walkable flags per tile. World-to-tile coordinate conversion. Viewport culling for efficient rendering.

SPH Fluid Simulation

Smoothed Particle Hydrodynamics with Poly6, Spiky, and Viscosity kernels. Configurable rest density, stiffness, viscosity, and surface tension. Interactive forces (push/attract). SpatialHashGrid for O(n) neighbor queries.

A* Pathfinding

Grid-based pathfinding with 4 or 8-directional movement. Manhattan/diagonal heuristics. Walkability map with obstacle marking. Path reconstruction. Random walkable tile selection for spawning.

Input Management

Unified keyboard, mouse, and wheel input. Per-frame state tracking: isKeyDown, isKeyPressed, isKeyReleased. Mouse position and delta. Event emission for reactive patterns.

Object Pooling

Generic ObjectPool<T> to minimize GC pressure. Factory and reset callbacks. Configurable initial and max size. Used by particles, fluid, and projectiles. prewarm() for initialization.

HiDPI Rendering

Automatic devicePixelRatio scaling for crisp retina rendering. Canvas abstraction with primitives: fillCircle, strokeCircle, fillRect, line, text. Radial gradients, shadows, and composite operations.

Quick Start

import { GameCore, Scene, Renderer } from './engine';

// Create your game scene
class MyScene extends Scene {
  update(dt: number) {
    // Game logic here
  }

  render(renderer: Renderer) {
    renderer.clear('#0a0a1a');
    renderer.fillCircle(400, 300, 50, '#6478ff');
  }
}

// Initialize and start
const game = new GameCore({
  canvas: document.getElementById('canvas'),
  fixedDt: 1 / 60
});
game.start(new MyScene());

Collision Detection

import { RigidBody } from './engine/physics/RigidBody';
import { CircleShape, AABBShape, OBBShape, PolygonShape } from './engine/physics/Shape';
import { CollisionDetector } from './engine/physics/CollisionDetector';

// Create bodies with different shapes
const player = new RigidBody(100, 100, 1, 'dynamic');
player.setShape(new CircleShape(25));

const wall = new RigidBody(200, 100, 0, 'static');
wall.setShape(new AABBShape(50, 100)); // halfW, halfH

const rotatedBox = new RigidBody(300, 100, 2, 'dynamic');
rotatedBox.setShape(new OBBShape(40, 20));
rotatedBox.angle = Math.PI / 4; // 45 degrees

// Test for collision
const manifold = CollisionDetector.test(player, wall);

if (manifold) {
  // Collision detected!
  console.log('Depth:', manifold.depth);
  console.log('Normal:', manifold.normal);
  console.log('Contacts:', manifold.contacts);
  
  // Separate bodies
  player.position.x += manifold.normal.x * manifold.depth;
  player.position.y += manifold.normal.y * manifold.depth;
}

Rigid Body Physics

import { RigidBody } from './engine/physics/RigidBody';
import { OBBShape } from './engine/physics/Shape';

// Create a dynamic rigid body
const box = new RigidBody(400, 100, 5, 'dynamic');
box.setShape(new OBBShape(30, 20));

// Configure material properties
box.material.restitution = 0.6;     // Bounciness (0-1)
box.material.staticFriction = 0.5;
box.material.kineticFriction = 0.3;
box.material.linearDamping = 0.1;
box.material.angularDamping = 0.05;

// Apply forces
box.applyForce(new Vector2(0, 500));  // Gravity
box.applyTorque(100);                  // Spin

// Apply impulse at a point (creates spin)
box.applyImpulseAt(
  new Vector2(50, 0),                // Impulse direction
  new Vector2(430, 80)               // World point
);

// Lock rotation (useful for characters)
box.lockRotation();

// Integrate physics (in update loop)
box.integrate(dt);

Object Pooling

import { ObjectPool } from './engine/commons/ObjectPool';

// Define pooled object
class Bullet {
  x = 0; y = 0; vx = 0; vy = 0; active = false;
  reset() { this.x = this.y = this.vx = this.vy = 0; this.active = false; }
}

// Create pool with factory and reset callbacks
const bulletPool = new ObjectPool<Bullet>(
  () => new Bullet(),     // Factory
  (b) => b.reset(),       // Reset callback
  100,                    // Initial size
  1000                    // Max size
);

// Prewarm pool at startup
bulletPool.prewarm(50);

// Acquire from pool (zero allocation)
const bullet = bulletPool.acquire();
bullet.x = playerX;
bullet.y = playerY;
bullet.vx = 500;
bullet.active = true;

// Return to pool when done
bulletPool.release(bullet);

// Check available objects
console.log(bulletPool.available);
